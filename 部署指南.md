# 🚀 部署到 Netlify（最简单，推荐）

## 需要上传的文件：
- `skin-calendar-new.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 脚本文件
- `index.html` - 自动跳转页
- `netlify.toml` - 配置文件

## 步骤：

### 1. 访问 Netlify
打开浏览器，访问：https://app.netlify.com/drop

### 2. 拖拽文件夹
- 将整个文件夹（包含上述5个文件）
- 直接拖拽到网页上的虚线框内
- 或者点击"browse to upload"选择文件夹

### 3. 等待部署
- 几秒钟后会自动部署完成
- 会生成一个链接，例如：https://random-name-123456.netlify.app

### 4. 访问网站
- 复制生成的链接
- 在手机或电脑浏览器打开
- 添加到手机主屏幕，像APP一样使用

### 5. 自定义域名（可选）
- 点击 "Site settings" → "Change site name"
- 改成你喜欢的名字，例如：my-skin-trading
- 新链接：https://my-skin-trading.netlify.app

# 🚀 部署到 GitHub Pages（免费，稳定）

## 步骤：

### 1. 创建 GitHub 账号
如果没有账号，访问：https://github.com/signup

### 2. 创建新仓库
- 点击右上角 "+" → "New repository"
- 仓库名：skin-trading-calendar
- 选择 "Public"
- 点击 "Create repository"

### 3. 上传文件
- 点击 "uploading an existing file"
- 拖拽以下文件：
  - skin-calendar-new.html
  - style.css
  - script.js
  - index.html
  - netlify.toml
- 点击 "Commit changes"

### 4. 开启 GitHub Pages
- 进入仓库的 "Settings"
- 左侧菜单找到 "Pages"
- Source 选择 "main" 分支
- 点击 "Save"

### 5. 访问网站
- 几分钟后，会显示链接：
- https://你的用户名.github.io/skin-trading-calendar/

# 🚀 部署到 Vercel（免费，快速）

## 步骤：

### 1. 访问 Vercel
打开：https://vercel.com/

### 2. 注册/登录
- 可以用 GitHub 账号登录

### 3. 导入项目
- 点击 "Add New" → "Project"
- 选择 "Import Git Repository" 或直接上传文件夹

### 4. 部署
- 点击 "Deploy"
- 等待几秒钟

### 5. 访问网站
- 会生成链接：https://your-project.vercel.app

# 📱 添加到手机主屏幕

部署完成后，可以将网站添加到手机主屏幕，像APP一样使用：

## iOS (iPhone/iPad)
1. 在 Safari 浏览器打开网站
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

## Android
1. 在 Chrome 浏览器打开网站
2. 点击右上角菜单（三个点）
3. 选择"添加到主屏幕"
4. 点击"添加"

# ⚠️ 重要提示

- **数据保存**：数据保存在浏览器本地存储（localStorage）
- **多设备同步**：不同设备的数据是独立的
- **备份数据**：建议定期使用"导出日历"功能备份
- **隐私安全**：所有数据都在你的浏览器本地，不会上传到服务器

# 🎯 推荐方案

**最简单**：Netlify（拖拽即可，无需注册）
**最稳定**：GitHub Pages（需要GitHub账号）
**最快速**：Vercel（需要注册）

我个人推荐 **Netlify**，因为：
✅ 无需注册即可部署
✅ 拖拽文件即可完成
✅ 自动HTTPS
✅ 全球CDN加速
✅ 可以随时更新
