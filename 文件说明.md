# 📁 文件说明

## 核心文件（必须上传）

### 1. skin-calendar-new.html
- 主页面文件
- 包含页面结构和布局

### 2. style.css
- 样式文件
- 包含所有的美化样式
- 响应式设计（PC和手机自适应）

### 3. script.js
- 脚本文件
- 包含所有功能逻辑
- 数据存储和管理

### 4. index.html
- 入口文件
- 自动跳转到主页面

### 5. netlify.toml
- Netlify配置文件
- 设置路由规则

## 说明文件（可选）

### README.md
- 项目说明
- 本地使用指南

### 部署指南.md
- 详细的部署步骤
- 多种部署方式

### 文件说明.md
- 当前文件
- 文件结构说明

## 部署到 Netlify

只需要上传这5个核心文件：
1. skin-calendar-new.html
2. style.css
3. script.js
4. index.html
5. netlify.toml

直接拖拽整个文件夹到 https://app.netlify.com/drop 即可！
